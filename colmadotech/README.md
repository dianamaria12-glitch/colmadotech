- 📈 **Reportes** por categoría e historial completo

## 🚀 Instalación

### Requisitos
- Python 3.8+
- pip

### Pasos

```bash
# 1. Clonar el repositorio
git clone https://github.com/tu-usuario/colmadotech.git
cd colmadotech

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar la aplicación
python app.py
```

### Acceder
Abre tu navegador en: `http://localhost:5000`

Si está en una VM, accede desde el host con: `http://[IP-DE-LA-VM]:5000`

## 🖥️ Stack Tecnológico
- **Backend:** Python + Flask
- **Base de datos:** SQLite
- **Frontend:** HTML5, CSS3, JavaScript vanilla
